# Hair Saloon

This app allows the user to
- Login/SignUp with the firebase mobile authentication.
-create profile using name,location(google map),image(using camera or file system)
-Booking an appointment by using different type of hair cut services.
- Create a profile and add the avatar image using camera or gallery.
-Admin page to acces all bookings
-cancel or accept page control by Admin


# It Demonstrates the following features
-Google map, activities, fragments, Firebase authentication and firestore.
-mobile camera

This app was written with a MinSdk of 21
