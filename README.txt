# Hair Saloon

This app allows the user to
- Login/SignUp with the firebase mobile authentication.
- Create a profile and add the avatar image using camera or gallery.
- Can determine the current location using Google maps, GeoCoding and GPS.
- View all the available services and its availabilities.
- Booking an appointment with any service and provide the availability to cancel.
- Track the status of the reserved appointments [accepted - rejected].


# It Demonstrates the following features
- Android activities, Intents, fragments and layouts.
- Local storage "Room" and Firebase for Auth.
- Google play services, Maps, GeoCoding.
- Mobile Camera and Gallery.
- RxJava, RxAndroid for asynchronous operations.
- Live Data.


This app was written with a MinSdk of 21
